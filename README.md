# DEPI Graduation Project – OpenCart E-Commerce Testing

## Project Overview
This is a team-based graduation project for the Digital Egypt Pioneers Initiative (DEPI) – Software Testing Track. 
The project focuses on testing an **OpenCart E-Commerce website**, covering Manual Testing, Automation Testing, and Database Testing.

**Team Size:** 6 members

## My Contributions
- Manual Testing:
  - Designed test cases for **Cart Page** and executed positive & negative test cycles.
  - Performed Manual Execution using **Localhost (XAMPP)**.
  - Logged and tracked bugs using **Zephyr** and reported them on **Jira**.
- Automation Testing:
  - Developed automated test scripts for **Cart Page** and **Home Page** using **Selenium WebDriver (Java)**, **TestNG**, and **Cucumber (BDD)**.
  - Worked on **Customer Page** in Admin Panel.
- Database Testing:
  - Validated backend data using **MySQL** queries.

## Tools & Technologies
- Selenium WebDriver (Java)
- TestNG
- Cucumber (BDD)
- Jira & Zephyr
- MySQL
- Postman (for API Testing)
- XAMPP (Localhost)
